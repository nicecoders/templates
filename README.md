## 快速上手

### 1. 安装相关依赖

```js
// 推荐
yarn global add @nicecode/cli

// or
// npm install @nicecode/cli -g
```

### 2. 运行命令

```js
// 查看脚手架版本号，是否安装成功
nice - V
```

### 3. 创建项目

```js
nice create ${项目名称}
```
